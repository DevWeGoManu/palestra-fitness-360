# Checklist Manuale Mobile

Eseguire questa checklist su dispositivi reali prima del go-live.

## Android Chrome

- Aprire URL staging in Chrome.
- Verificare caricamento iniziale senza errori console.
- Login admin.
- Logout.
- Login istruttore.
- Creazione utente atleta.
- Creazione programma allenamento.
- Aggiunta esercizi da libreria.
- Drag & drop esercizi.
- Salvataggio programma.
- Login atleta.
- Visualizzazione programma assegnato.
- Pulsante "Completa allenamento".
- Pagina Storico.
- Export PDF/Stampa.
- Installazione PWA da prompt Chrome.
- Avvio PWA installata.
- Test offline base: aprire app shell senza rete dopo primo caricamento.

## iPhone Safari

- Aprire URL staging in Safari.
- Verificare layout mobile e safe area.
- Login/logout.
- Form creazione utente con tastiera mobile.
- Form editor programma.
- Sticky actions non coprono i campi.
- Scroll day/esercizi fluido.
- Completamento allenamento.
- Storico.
- Stampa/Condivisione PDF da dialog Safari.
- Aggiungi alla schermata Home.
- Avvio dalla Home.
- Verificare sessione dopo chiusura/riapertura.

## Landscape

- Ruotare dispositivo.
- Verificare sidebar mobile e scroll.
- Verificare editor programma.
- Verificare sticky actions.

## Accessibilita rapida

- Navigazione con tastiera esterna se disponibile.
- Focus visibile.
- Target touch comodi.
- Test zoom testo/browser.
